Description

Hardware and networking comprise a logical interlink of several computer parts and systems. This interlink can be either on a small-scale basis or on a large-scale basis. When you associate computers with scanners, printers, modems, routers etc., it becomes small scale interconnection. When there is an interlink of a very huge set of computer machines with each other and with the database and servers then it is large scale interconnection.
