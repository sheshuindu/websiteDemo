Description

The Ab Initio software is a Business Intelligence platform containing six data processing products: Co>Operating System, The Component Library, Graphical Development Environment, Enterprise Meta>Environment, Data Profiler and Conduct>It. It is a powerful graphical user interface-based parallel processing tool for ETL data management and analysis. Ab Initio Co>Operating System is a foundation for all Ab Initio applications and provides a general engine for integration of all kinds of data processing and communication between all the tools within the platform. It runs on OS/390, zOS on Mainframe, Unix, Linux, and Windows. It enables distributed and parallel execution, platform-independent data transport, establishing checkpoints and monitoring of the process. It implements data execution parallelism by using data parallelism, component parallelism and pipeline parallelism.
